# Oatty Site

Static Lit + TypeScript site for https://oatty.io.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes

- CSS follows SMACSS layering in `src/styles/css/`.
- Components consume reusable constructible stylesheets from `src/styles/sheets/`.
- Theme mode supports `system`, `light`, `dark`, and `high-contrast`.
